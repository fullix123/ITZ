# ИТЗ
Dark fantasy game


Чтобы начать игру, запустите файл start_game_here.py



В игре реализовано пару интерестных механик:
1.У каждого оружия есть свой модификатор
2.В битве с боссом нельзя использовать 1 оружие 2 раза подряд
3.Интестные игры с мини боссами на локациях

Остальные механики были взяты из примера
